VNS.Gallery Plugin - Demos

A flexible jQuery gallery plugin with thumbnails, lightbox and grid view

Demo 1: Basic Usage

The simplest setup - just add images in a div and initialize the plugin. Drag the carousel with your mouse or swipe with your finger to navigate!

<div class="myGallery">
	<img src="img1.jpg" alt="Image 1">
	<img src="img2.jpg" alt="Image 2">
	<img src="img3.jpg" alt="Image 3">
	...
</div>

<script>
	var gallery = $('.myGallery').vnsGallery();
</script>

Demo 2: With Loop & Touch/Drag Support

Enable loop to navigate in a circle through both thumbnails and lightbox. Touch/drag support is enabled by default - try dragging the carousel with your mouse or finger!

var gallery = $('.gallery-loop').vnsGallery({
	loop: true,
	step: 2,  // Step 2 images at a time
	enableDrag: true,  // Enable drag/swipe (default: true)
	dragThreshold: 50  // Minimum drag distance in pixels (default: 50)
});

Demo 3.5: Disable Drag Navigation

If you prefer to only use the navigation buttons, you can disable drag/swipe functionality:

var gallery = $('.gallery-no-drag').vnsGallery({
	enableDrag: false  // Disable drag/swipe navigation
});

Demo 4: Event Handlers

Attach event handlers to different events:

var gallery = $('.gallery-events').vnsGallery({
	loop: true,
	onShow: function() {
		console.log('Lightbox opening...');
	},
	onShown: function() {
		console.log('Lightbox is open');
	},
	onChange: function(index) {
		console.log('Changing to image: ' + (index + 1));
	},
	onChanged: function(index, image) {
		console.log('Now showing image: ' + (index + 1));
	},
	onClose: function() {
		console.log('Closing lightbox...');
	},
	onClosed: function() {
		console.log('Lightbox closed');
	}
});

// You can also listen to events via jQuery:
$('.gallery-events').on('show', function() {
	console.log('Event: show');
});

Demo 5: Public Methods

Use public methods to control the gallery programmatically:

var gallery = $('.gallery-methods').vnsGallery();

// Open lightbox with image #2
gallery.open(1);

// Open grid view
gallery.open();

// Close lightbox
gallery.close();

// Next image
gallery.next();

// Previous image
gallery.prev();

// Refresh gallery (e.g. after AJAX)
gallery.refresh();

// Destroy gallery
gallery.destroy();
Tip: Once the modal is open, use these methods:
  • Next/Previous: Use arrow keys or click navigation buttons in the modal
  • Close: Press ESC key or click the X button in the modal
  • API calls: You can also call gallery.next(), gallery.prev(), gallery.close() from browser console

Demo 6: Static Grid (No Carousel)

Display images in a static grid without carousel navigation:

var gallery = $('.gallery-static').vnsGallery({
	useCarousel: false,
	columns: 4,
	maxImages: 6,
	showMoreIndicator: true,
	moreIndicatorAction: 'load'  // or 'modal'
});

Demo 7: Many Columns on Front Page (10 Columns)

Show many small thumbnails on the front page with columns set to 10:

var gallery = $('.gallery-front-columns').vnsGallery({
	useCarousel: false,
	columns: 10
});

Demo 8: Many Columns in Modal (10 Columns)

Show many small thumbnails in modal with modalColumns set to 10:

var gallery = $('.gallery-many-columns').vnsGallery({
	modalColumns: 10,
	modalWidth: '95vw'
});

Demo 9: Thumbnail vs Fullsize Images

Use data-fullsize attribute to specify a larger image for single view. Carousel and grid use thumbnails for performance, single view loads fullsize:

<img src="thumbnail.jpg" data-fullsize="fullsize.jpg" alt="Image">

Options Reference

Image Attributes

You can use the following data attributes on your <img> tags:

  • data-fullsize - URL to a larger/higher quality image that will be loaded in single view. If not specified, the src will be used.

Example:

<img src="thumbnail-small.jpg" data-fullsize="photo-large.jpg" alt="Photo">

This improves performance by loading small thumbnails for the carousel and grid, then loading full-size images only when viewing a single image.

Plugin Options

Option Type Default Description
useCarousel Boolean true Use carousel navigation (if false, shows static grid)
loop Boolean false Enable looping through images (carousel only)
step Number/null null Number of images to step at a time (null = auto based on visible, carousel only)
maxImages Number/null null Max images to show initially in static grid (null = show all)
showMoreIndicator Boolean true Show "..." indicator when images are hidden in static grid
moreIndicatorAction String 'modal' Action on indicator click: 'modal' (open gallery) or 'load' (load more)
moreIndicatorText String '...' Text for the more indicator
columns Number/null null Number of columns in front (null = use responsive breakpoints)
modalColumns Number 4 Number of columns in modal grid view (2-6)
showAllButton Boolean true Show "See all" button
showNavigation Boolean true Show prev/next arrows in lightbox
showCounter Boolean true Show image counter
showCloseButton Boolean false Show close (X) button in modal
enableKeyboard Boolean true Enable keyboard navigation (arrow keys and ESC)
showAllText String 'Show all' Text for "Show all" button
prevText String 'â—€' Text/HTML for previous button
nextText String 'â–¶' Text/HTML for next button

Event Callbacks

Event Description
onShow Called before lightbox opens
onShown Called after lightbox is opened
onChange Called before image changes
onChanged Called after image is changed
onClose Called before lightbox closes
onClosed Called after lightbox is closed
onNext Called before next image is shown
onNextDone Called after next image is shown
onPrev Called before previous image is shown
onPrevDone Called after previous image is shown

Public Methods

Method Description
open(index) Open lightbox (with optional image index)
close() Close lightbox
next() Go to next image
prev() Go to previous image
destroy() Destroy gallery instance
refresh() Refresh gallery (useful after DOM changes)

jQuery Events

You can also listen to events via jQuery's standard event system:

var gallery = $('.myGallery').vnsGallery();

$('.myGallery').on('show', function() { ... });
$('.myGallery').on('shown', function() { ... });
$('.myGallery').on('change', function(e, index) { ... });
$('.myGallery').on('changed', function(e, index, image) { ... });
$('.myGallery').on('close', function() { ... });
$('.myGallery').on('closed', function() { ... });
$('.myGallery').on('next', function(e, index) { ... });
$('.myGallery').on('prev', function(e, index) { ... });
